***Úkol č.1***

**Zadání:**
Napiš Python skript, který zavolá LLM API, použije nástroj (např. výpočetní funkci) a vrátí odpověď zpět LLM.

Skript využívá LLM API: Gemini – AIStudio

- Je nutné mít vygenerovaný GEMINI_API_KEY, který je v souboru .env

- Před spuštěním kódu si nainstaluj příslušné knihovny a vytvoř si virtuální prostředí 

Create a virtual environment

```bash
uv venv
```

Activate virtual environment

```bash
source .venv/bin/activate
```

Install packages

```bash
uv sync
```

Run script 

```bash
python 1_ukol_soucet.py
```
